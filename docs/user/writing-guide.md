# 写作指南

本指南介绍如何在 FNW 中高效写作。

## 智能引用

在写作过程中，您可以快速引用已创建的角色、地点等实体：

### 引用角色
```
@林逸 踏入山门，心中暗暗发誓...
```
自动转换为：`[[角色:林逸]]`

### 引用地点
```
@青云山 是这一带最大的修仙门派...
```
自动转换为：`[[地点:青云山]]`

### 引用功法
```
他修炼的@青元剑诀 已经小成...
```

## Markdown 支持

编辑器支持完整的 Markdown 语法：

```markdown
# 大标题
## 二级标题
### 三级标题

**粗体** *斜体* ~~删除线~~

- 列表项1
- 列表项2

1. 有序列表
2. 有序列表

> 引用

```
代码块
```

| 表格 | 表头 |
|------|------|
| 内容 | 内容 |
```

## 沉浸式写作

启用沉浸模式：
1. 点击编辑器右上角「设置」图标
2. 开启「沉浸模式」
3. 隐藏所有UI元素，专注写作

## 打字机模式

开启后，当前编辑行会保持在屏幕中央：

1. 设置 → 打字机模式 → 开启

## 自动保存

- 默认每30秒自动保存
- 手动保存：Ctrl+S
- 查看保存状态：状态栏

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+S | 保存 |
| Ctrl+N | 新建章节 |
| Ctrl+F | 查找 |
| Ctrl+H | 替换 |
| Ctrl+Z | 撤销 |
| Ctrl+Y | 重做 |
| Ctrl+/ | 注释 |
