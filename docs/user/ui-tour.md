# 界面导览

本指南介绍 Fantasy Novel Workshop 的主要界面区域。

## 主界面布局

```
┌─────────────────────────────────────────────────────┐
│  标题栏                              [_] [□] [X]   │
├────────┬────────────────────────────────────────────┤
│        │                                            │
│  导航栏  │              主内容区                    │
│        │                                            │
│ • 项目  │  根据选择显示不同内容                       │
│ • 写作  │                                            │
│ • 角色  │                                            │
│ • 世界  │                                            │
│ • 统计  │                                            │
│ • AI   │                                            │
│        │                                            │
├────────┴────────────────────────────────────────────┤
│  状态栏：字数 | 保存状态 | 当前章节                  │
└─────────────────────────────────────────────────────┘
```

## 导航栏功能

### 1. 项目管理
- 项目列表展示
- 创建/删除/导入项目

### 2. 写作编辑器
- 章节列表
- Monaco Editor 编辑器
- 预览模式

### 3. 角色管理
- 角色卡片列表
- 关系图可视化（G6）

### 4. 世界观构建
- 修炼体系（境界/功法/技能）
- 地理设定（地点/势力）
- 时间线（纪元/事件）

### 5. 统计面板
- 字数统计
- 写作速度图表
- 目标进度

### 6. AI 助手
- Ollama 本地AI
- 写作增强
- 一致性检查

## 工具栏

| 图标 | 功能 |
|------|------|
| 📁 | 新建项目 |
| 💾 | 保存 |
| 📤 | 导出 |
| ⚙️ | 设置 |

## 快捷操作

- **右键菜单**：在角色/章节上右键显示操作菜单
- **拖拽**：调整章节顺序、角色关系
- **双击**：编辑角色/章节
