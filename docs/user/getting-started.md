# 快速开始指南

欢迎使用 Fantasy Novel Workshop (FNW)！本指南将帮助您在5分钟内创建第一个项目。

## 第一步：创建项目

1. 启动应用程序
2. 点击主界面上的「新建项目」按钮
3. 输入项目名称（如：我的修仙小说）
4. 选择项目模板或从空白项目开始
5. 点击「创建」

## 第二步：构建世界观

创建项目后，系统会自动跳转到世界观编辑器。

### 设计修炼体系

1. 切换到「修炼体系」标签
2. 点击「添加境界」
3. 输入境界名称、描述、序号
4. 常见境界：炼气期 → 筑基期 → 金丹期 → 元婴期 → 化神期

### 创建功法

1. 在「功法」区域点击「添加功法」
2. 设定功法名称、类型、适用境界
3. 功法类型：攻击型、防御型、辅助型

## 第三步：创建角色

1. 点击左侧导航栏的「角色」
2. 点击「新建角色」
3. 填写角色信息：
   - 姓名（必填）
   - 境界、功法
   - 角色定位：主角/配角/反派
4. 使用关系图连接角色

## 第四步：开始写作

1. 点击「写作」进入编辑器
2. 创建章节
3. 使用 Markdown 语法写作
4. 使用 `@角色名` 引用角色
5. 使用 `#标签` 添加标签

## 快捷键

| 功能 | 快捷键 |
|------|--------|
| 保存 | Ctrl+S |
| 新建章节 | Ctrl+N |
| 搜索 | Ctrl+F |
| 替换 | Ctrl+H |
