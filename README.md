# Fantasy Novel Workshop (FNW)

纯本地玄幻小说写作系统

## 简介

FNW 是一个专为玄幻小说作者设计的本地写作工具，完全离线运行，保护您的创作数据安全。

## 功能特性

- 📚 **项目管理** - 多作品管理，支持切换
- 📝 **大纲管理** - 故事大纲、结构规划、章节大纲
- 👥 **角色管理** - 角色档案、人物关系图
- 🌍 **世界观设定** - 修炼体系、法宝、势力、地理
- ✍️ **正文写作** - Markdown 编辑器、章节管理
- 📊 **写作统计** - 字数统计、进度追踪
- 💾 **本地存储** - SQLite 数据库，纯本地运行

## 技术栈

- Electron 28
- React 18 + TypeScript
- Zustand (状态管理)
- Ant Design (UI组件)
- sql.js (本地数据库)
- electron-vite (构建工具)

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build

# 打包发布
npm run package
```
